==========
ImageToPng
==========


----
説明

画像ファイルを PNG 形式に変換する。
フォルダを指定した場合、配下にある全ての画像ファイルを処理する。

対象ファイル・フォルダの後に編集オプションを指定できる。


----
コマンド

ImageToPng.exe ( 対象ファイル | 対象フォルダ ) [/E 幅 高さ] [/C 幅 高さ エリア]


----
編集オプション

/E 幅 高さ

	拡大・縮小を行う。

/C 幅 高さ エリア

	エリアを指定して切り出す。

	エリア：
		1 == 左下
		2 == 下
		3 == 右下
		4 == 左
		5 == 中央
		6 == 右
		7 == 左上
		8 == 上
		9 == 右上
